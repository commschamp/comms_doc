#include "comms/field/NoValue.h"

Detailed Description

template<typename TFieldBase>
class comms::field::NoValue< TFieldBase >

Dummy field with "do-nothing" read/write operations.

Can be used with other classes that require field types.

Template Parameters
TFieldBaseBase class for this field, expected to be a variant of comms::Field.
