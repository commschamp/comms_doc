Mode to be used by comms::field::Optional.

Enumerator
Tentative 

The field existence is tentative, i.e. If there is enough bytes to read the field's value, than field exists, if not then it doesn't exist.

Exists 

Field must exist.

Missing 

Field doesn't exist.

NumOfModes 

Number of possible modes, must be last.
